## Build & Run Instructions

You need a C++ compiler (e.g., g++, clang++) installed. To compile and run an exercise:

```
g++ -o Course.exe Course.cpp
./Course.exe
```

Replace `Course.cpp` and `Course.exe` with the file you want to build/run.
